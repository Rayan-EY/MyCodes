About Our Contact List Webpage

This webpage is a Contact List application with the following features:

  • Navigation Buttons: At the top of the page, there are two buttons, "Home" and "About." These buttons allow users to navigate to the home page or the about page respectively.

  • Contact Form: In the main content area, there is a contact form. Users can input their contact information, including Name, Email, and Phone number. Each field is marked as required to ensure users provide essential information.

  • Add Contact Button: Below the contact form, there is an "Add Contact" button. Clicking this button triggers the addition of a contact to the contact list.

  • Contact List: On the right-hand side, there is a section titled "Contact List." This is where added contacts are displayed. Initially, it appears empty, but as users add contacts, they will be listed here.

  • Search: Below the contact list, there is a search feature. Users can input a name into the "Search by Name" field and click the "Search" button to filter and display contacts with matching names.

  • Sort: Below the search feature, there is a "Sort Alphabetically" button. Clicking this button sorts the displayed contacts in alphabetical order by name.

  • JavaScript: The functionality of the page is powered by JavaScript. Functions like addContact(), searchContacts(), and sortContacts() are called when their respective buttons are clicked, allowing dynamic interaction with the contact list.

About Time Complexities

Linear Search

Linear search has a time complexity of O(n), where n is the number of elements in the list. It searches for a specific element by iterating through the list one element at a time until the target element is found or the end of the list is reached.

Bubble Sort

Bubble sort has an average-case and worst-case time complexity of O(n^2), where n is the number of elements in the list. It repeatedly compares adjacent elements and swaps them if they are in the wrong order until the entire list is sorted.
